# Dev-Rev-Covid-Vaccination
Web application for vaccination slot booking
Create a web application for Covid Vaccination

Tech Stack:
Front-end : Any
Back-end : Any
Db : Any

Types of Users:
User
Admin

User Use cases:
- login
- signup
- search for Vaccination centre
- apply for vaccination (only 10 candidates allowed per day)
- logout

Admin Use Cases
- login (for admin use separate UI)
- Add Vaccination centres
- Get dosage details (group by centres)
- Remove vaccination centres
